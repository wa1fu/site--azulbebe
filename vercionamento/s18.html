1.Criptografia

é o processo de codificar informações para torná-las seguras. Ela transforma textos simples em códigos abstratos, conhecidos como textos cifrados, que só podem ser lidos com uma chave específica. A criptografia é usada para proteger dados em repouso, como arquivos armazenados em um computador, e dados em trânsito, como informações enviadas pela internet.

2. O protocolo TCP/IP

é um conjunto de protocolos que possibilita a comunicação entre computadores e servidores, formando a internet. Ele define como as informações se movimentam de um remetente para um receptor, garantindo a transmissão precisa e correta dos dados. O TCP/IP é um modelo de comunicação que combina o Protocolo de Controle de Transmissão (TCP) com o Protocolo de Internet (IP).

3.Message Queuing (Filas de Mensagens)

é um componente de software usado para a comunicação entre processos ou threads. Ele permite que aplicativos e serviços independentes troquem informações de forma assíncrona, ou seja, sem que os endpoints que produzem e consomem mensagens precisem interagir ao mesmo tempo. As filas de mensagens são usadas em arquiteturas sem servidor e de microsserviços.

4.RabbitMQ e Apache Kafka

são softwares de mensagens que implementam o protocolo "Advanced Message Queuing Protocol". O RabbitMQ é um software de mensagens com código aberto, que foi estendido com uma arquitetura de plug-in para suportar o protocolo "Streaming Text Oriented Messaging Protocol". O Apache Kafka é uma plataforma open-source de processamento de streams desenvolvida pela Apache Software Foundation, escrita em Scala e Java. Ele fornece uma plataforma unificada, de alta capacidade e baixa latência para publicar, subscrever, armazenar e processar fluxos de dados.

5.Throughput

é a taxa de transferência efetiva de um sistema. Ele representa a quantidade de dados que são transmitidos em um determinado período de tempo. O throughput pode ser usado para medir a eficiência de uma rede, um servidor ou um aplicativo.
